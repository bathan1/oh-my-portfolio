Projects

BioMatch

Made for the BGB Group Prompt at HopHacks 2023, where the prompt was to build a piece of software that simplified connecting underrepresented populations to clinical trials. Our solution was to make a clinical trials finder that matches users with clinical trials that are most relevant to them. Made with 3 other group members, we built this using ReactJS, ExpressJS, and MongoDB.

Manga-Cover App

A project made for my Fullstack Web Dev course, where we built a web page that interacts with OpenAI API to generate a summary and cover image for a manga based on the title and the theme of the page.

Sticky-Notes App

Another project for my Fullstack Web Dev course, where I made a sticky notes app using vanilla HTML/CSS/JavaScript that allows user to write, save, edit, and delete sticky notes quickly.

Calculator App

One of the first websites I've ever made with a simple goal of being able to run a basic functional calculator that made its calculations on the client-side. This was my first introduction to building websites ever, and is what kickstarted my passion for web development.
